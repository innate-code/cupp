источник статьи
1. Crunch - утилита для создания простых словарей. 
К примеру, нам нужен словарь состоящий из цифр от 0 до 9 , длина пароля варьируется от 6 до 8. 
Для этого мы переходим в наш любимы термукс, и устанавливаем данную утилиту:
pkg update
pkg upgrade
pkg install crunch
Теперь нам понадобится запомнить одну команду :
crunch <минимальная-длина> <максимальная-длина> [набор символов]
Т.е для решения нашей задачи, команда будет такой :
crunch 6 9 0123456789
Готово , нам на экран выводятся списки паролей, их можно скопировать и вставить в текстовый документ, но это долго, поэтому допишем аргумент к команде:
>>file  - сохранение в какой либо файл
crunch 6 9 0123456789 >>file
p.s для просмотра всех команд пропишите : crunch -h
Готово, теперь у нас есть словарь, который мы можем в дальнейшем использовать. Но что делать если пароль состоит не только из цифр но и из каких либо слов? Так же можно выполнить такую задачу в crunch указав вместо цифр, буквы. Но что если мы знаем что пароль содержит какую-то личную информацию? Для этого нам поможет утилита Cupp
2. Cupp - утилита которая создает пароли по вашим грубо говоря предпочтениям. Точнее по предпочтениям жертвы. Т.е ФИО жертвы, дата рождения, увлечения и тд.
Установка :
apt update 
apt upgrade -y
apt install git
apt install python
git clone https://github.com/Mebus/cupp.git
cd cupp
chmod +x cupp.py
python cupp.py
Использование :
python cupp.py -i
После чего вам будет представлена форма которую необходимо заполнить :
написал перевод
После того как вы заполните форму, пройдет генерация словаря, и он сохранится в директории cupp.
